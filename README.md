# MedXtorePharmacyV2

## Danh Mục

`Web bán hàng` `Full Stack`

## Giới Thiệu

MedXtorePharmacyV2 là phiên bản nâng cấp của hệ thống quản lý nhà thuốc MedXtorePharmacy (POS), được cải thiện về kiến trúc và tính năng.

## Chức Năng

- Hệ thống POS bán thuốc
- Quản lý kho thuốc
- Trang quản trị admin
- API cho ứng dụng khách
- Quản lý đơn hàng và hóa đơn

## Công Nghệ Sử Dụng

- **Backend**: PHP, Composer
- **Cơ sở dữ liệu**: MySQL
- **Frontend**: HTML, CSS, JavaScript
- **Kiến trúc**: MVC

## Yêu Cầu Hệ Thống

- PHP 7.4 trở lên
- MySQL
- Composer
- Apache/Nginx

## Cài Đặt

1. Clone repository
2. Cài đặt dependencies:
```bash
composer install
```
3. Import database từ thư mục `database/`
4. Cấu hình kết nối trong `config/`

## Chạy Ứng Dụng

- Cấu hình web server trỏ vào thư mục project
- Truy cập qua trình duyệt
